Welcome to the Tidal Accelerator hands-on lab session! This page contains all the resources you'll need to complete the lab exercises.

## Required Software

Before starting the lab, you'll need to install:

1. [Tidal Tools](https://get.tidal.sh/) - Download and install Tidal Tools. For help see our documentation [here](tidal-tools.html).
2. [Docker](https://docs.docker.com/get-docker/) - Required for running Tidal Tools' code analysis feature.

{: .note }
Make sure both Tidal Tools and Docker are properly installed and running before proceeding with the lab exercises.

## Required Resources

For this hands-on lab session, you'll need to download the following resources:

### 1. Demo Resources CSV
This CSV file contains sample application data that you'll use during the lab session.
- [Download demo-resources.xlsx](https://s3.ca-central-1.amazonaws.com/tidal.assets.public/demo-data/demo-resources.xlsx)

### 2. SCT Results
This zip file contains sample Server Configuration Tool (SCT) results for analysis.
- [Download demo-sct-results.zip](https://s3.ca-central-1.amazonaws.com/tidal.assets.public/demo-data/demo-sct-results.zip)

### 3. Machine Statistics
This JSON file contains sample machine statistics data for analysis.
- [Download machine_stats.json](/assets/machine_stats.json)

{: .note }
Make sure to download all three resources before starting the lab session. You'll need them to complete the exercises.

## Getting Started

1. Download all three resources listed above
2. Extract the `demo-sct-results.zip` file to a location of your choice
3. Keep the CSV and JSON files in their original format

## Need Help?

If you encounter any issues downloading or using these resources, please let us know.
